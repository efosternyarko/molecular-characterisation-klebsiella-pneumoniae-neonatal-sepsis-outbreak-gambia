# Figure S2 — Transmission Cluster Plots (*K. pneumoniae* and *K. quasipneumoniae*)

Figure S2 shows transmission cluster summaries for *K. pneumoniae* (Kp) and
*K. quasipneumoniae* (Kqp) isolates, generated using the **transmission_estimator** tool.

---

## Tool

**transmission_estimator**  
- GitHub: https://github.com/klebgenomics/transmission_estimator  
- Shiny app: https://klebsiella.shinyapps.io/transmission_estimator/

---

## Inputs

### 1. SNP distance matrix

A pairwise SNP distance matrix computed from the core genome alignment.

**How to generate:**

```bash
# 1. Align reads to reference genome with Snippy (run per isolate)
snippy --outdir <sample_dir> --ref <reference.gbk> --R1 <reads_R1.fastq.gz> --R2 <reads_R2.fastq.gz>

# 2. Generate core genome alignment
snippy-core --ref <reference.gbk> <sample_dir_1> <sample_dir_2> ...

# 3. Compute pairwise SNP distances
snp-dists core.full.aln > snp_distance_matrix.tsv
```

Software versions used in this study: Snippy v4.6, snp-dists v0.8.2.

### 2. Metadata file

A tab- or comma-separated file with one row per isolate.

**Required columns:**

| Column | Description |
|--------|-------------|
| `id` | Isolate identifier (must match row/column names in the SNP matrix) |
| `Year` | Collection year |
| `Month` | Collection month (numeric) |
| `Day` | Collection day |
| `Country` | Country of isolation |
| `Site` | Hospital / sampling site |
| `ST` | MLST sequence type (from Kleborate output) |

Additional columns from Kleborate output (e.g. `K_locus`, `O_locus`, AMR gene columns) are passed through and displayed in the cluster summaries.

---

## Running the analysis

### Option A — Shiny web app (no installation required)

1. Go to https://klebsiella.shinyapps.io/transmission_estimator/
2. Upload the SNP distance matrix and metadata file
3. Set the SNP threshold (this study used **≤10 SNPs** to define putative transmission clusters)
4. Download the cluster assignment table and plots

### Option B — R package (command line)

```r
# Install
remotes::install_github("klebgenomics/transmission_estimator")

library(TransmissionEstimator)

# Load inputs
snp_matrix <- read.csv("snp_distance_matrix.tsv", sep = "\t", row.names = 1)
metadata   <- read.csv("metadata.csv")

# Run cluster analysis (SNP threshold = 10)
clusters <- assign_clusters(snp_matrix, threshold = 10)

# Merge with metadata and plot
results <- merge_metadata(clusters, metadata)
plot_clusters(results)
```

## Reproducing Figure S2

Figure S2 was generated by running the transmission_estimator on separate inputs
for *K. pneumoniae* and *K. quasipneumoniae* isolates, then combining the output
plots in the Shiny interface. The full methods are described in the paper
(Methods section: *Genomic cluster analysis*).

The SNP distance matrices and metadata files used as input are available from
the authors on request (e.foster-nyarko@lshtm.ac.uk).
