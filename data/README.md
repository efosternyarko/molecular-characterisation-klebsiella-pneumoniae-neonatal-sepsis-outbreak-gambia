# Data Files

This directory should contain the input data files required by the analysis scripts.
Most files are not included in this repository due to size or data sharing restrictions
but are available as described below.

---

## Files Required

### Figure 1A — `fig01A_gambia_map.py`

| File | Description | Source |
|------|-------------|--------|
| `gadm41_GMB_1.shp` (+ `.dbf`, `.prj`, `.shx`) | Administrative boundaries of The Gambia at regional level | Download from [GADM](https://gadm.org/download_country.html): select Gambia, level 1. Free for non-commercial use. |

---

### Figure 2 — `fig02_resistance_heatmap.py`

| File | Description | Source |
|------|-------------|--------|
| `kpn_clinical_AMR.csv` | Per-isolate AST data for clinical *K. pneumoniae* | Derived from Supplementary File S2 (published with paper) |
| `kpn_environmental_AMR.csv` | Per-isolate AST data for environmental *K. pneumoniae* | Derived from Supplementary File S2 (published with paper) |

**Column requirements:**  
Both CSVs: `sample_id`, `status` (used to flag ST39 isolates), antibiotic columns with R/I/S values.  
Environmental CSV additionally requires: `source_type` (e.g. "IV bag", "water").

### Figures 4, 5, S3 — epi curve, ST39 tree, ST39 plasmid

|------|-------------|--------|

## Notes

- Place all files directly in this `data/` directory (alongside this README).
- File paths in the scripts assume data files are in `data/` relative to the script (i.e. scripts and the `data/` directory sit side by side).
- The Gambia shapefile archive can be unzipped here; only `gadm41_GMB_1.shp` (and its companion files) are needed.
- For access to restricted data files, contact the corresponding author:
